# 时光轴标签体系

## 事件类型标签（按 `type` 字段）

| 标签 | CSS 类 | 颜色 | 数据来源 |
|------|--------|------|----------|
| 里程碑 | `.milestone` | 🟡 金色 `#fbbf24` | 手动 |
| 巡演 | `.tour` | 🟢 绿色 `#4ade80` | 手动 |
| 公演 | `.show` | 🟣 紫色 `#c084fc` | 手动、小助理 |
| 行程 | `.event` | 🔵 蓝色 `#60a5fa` | 手动 |
| 外务 | `.external` | 🟠 橙色 `#fb923c` | 手动、小助理 |
| 直播 | `.event` | 🔵 蓝色 `#60a5fa` | 房间 |
| 见面会 | `.event` | 🔵 蓝色 `#60a5fa` | 小助理 |
| 其他 | `.event` | 🔵 蓝色 `#60a5fa` | 小助理 |

## 标题关键词标签（覆盖类型标签）

当行程名称包含以下关键词时，**替换**原来的类型标签：

| 标签 | CSS 类 | 颜色 | 触发关键词 |
|------|--------|------|-----------|
| 巡演 | `.tour` | 🟢 绿色 `#4ade80` | 名称含"巡演" |
| 首演 | `.milestone` | 🟡 金色 `#fbbf24` | 名称含"首演" |
| 助演 | `.show` | 🟣 紫色 `#c084fc` | 名称含"助演" |

> 规则：关键词标签优先级高于类型标签，同时只显示一个。

## 其他状态标签

| 标签 | CSS 类 | 颜色 | 说明 |
|------|--------|------|------|
| 有弹幕 | `.danmu.available` | 🔵 蓝色 `#60a5fa` | 房间直播有弹幕文件 |
| 无弹幕 | `.danmu.missing` | ⚪ 灰色 `#9ca3af` | 房间直播无弹幕 |
| 回放 | `.replay` (inline) | 🟢 绿色 `#4ade80` | 有回放视频链接 |

## 筛选按钮

| 按钮 | 数据源标识 | 显示内容 |
|------|-----------|----------|
| 全部 | `all` | 手动 + 房间 + 小助理 |
| 手动 | `manual` | 仅手动事件 |
| 房间 | `room` | 仅房间直播 |
| 小助理 | `assistant` | 仅小助理行程 |
