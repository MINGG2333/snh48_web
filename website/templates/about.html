关于本站

项目目标

本网站旨在记录和整理喜爱的演艺人员的公开信息。通过整合网络上的公开视频资源， 利用 AI 技术（LLM）对视频转录文本进行分析和问答，让粉丝能够快速检索到 演艺人员在直播和节目中提到的重要信息。

技术架构

后端框架

FastAPI (Python)

向量数据库

ChromaDB

语义检索

text2vec + BM25

LLM 问答

DeepSeek / OpenAI API

功能特色

  • 全屏背景 + 动态滚动文本展示
  • 基于视频转录文本的 AI 智能问答
  • 向量 + BM25 混合检索，精准召回
  • LLM 分析筛选 + 引用溯源
  • 链接外部网站的多媒体资源
